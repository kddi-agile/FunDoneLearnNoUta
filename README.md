## Fun Done Learnを始めたい方はこちら
### 楽しいふりかえりでチームを成長させよう
[anycommu](https://anycommu.web.app)は、仕事やコミュニティ・家族・友人など、チームで活動するすべての人を支援するふりかえりツールです。同じ場所でもリモートでもハイブリットでも、気軽にサクッと楽しくふりかえり。定期的にふりかえりを行うことで、コミュニケーションを円滑にしチームの成長をお手伝いします。
